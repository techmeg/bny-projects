Grids 101

So, you have declared your element a grid. First, the basics: vertical sections are called "columns" and horizontal sections are called "rows." To design your grid, you must set some basic parameters, starting with grid-template-columns, grid-template-rows (optional), grid-gap, as well as other, more general styling.

For example, your html might look like this:

		<div class="grid-container">
				<div class="grid-item">1</div>
				<div class="grid-item">2</div>
				<div class="grid-item">3v/div>
				<div class="grid-item">4</div>
				<div class="grid-item">5</div>
				<div class="grid-item item-six">>6</div>
				<div class="grid-item item-seven">7</div>
				<div class="grid-item">8</div>
				<div class="grid-item">9</div>
				<div class="grid-item">10</div>
			</div>

and the CSS like this:

		
		.grid-container {
			display: inline-grid;
			grid-template-columns: 60px 60px 60px 60px;
			grid-gap: 10px;
			padding: 15px;
			background-color: skyblue;
		}
		.grid-item {
			background-color: violet;
			border: 1px solid darkblue;
			padding: 20px;
			color: white;
		}
		.item-six {
			grid-column: 3/5; <!-- alternatively, grid-column: 3 / span 2 -->
		}
		.item-seven {
			grid-row: 2/4;
			
		}

The grid-template-columns property is set with values for each column in px, ems, auto or names (more on that in Grids Advanced). Grid-gap is the shorthand to set the spacing between the columns and rows. Optionally, grid row height can be set using grid-template-row, or you can break specifications down further using grid-column-start, grid-column-end, grid-row-start, grid-row-end, grid-column-gap, grid-row-gap.

To illustrate, the code for the same grid might look like this:

		.grid-container {
			display: inline-grid;
			grid-template-columns: 60px 60px 60px 60px;
			grid-template-rows: 60px;
			grid-column-gap: 10px;
			grid-row-gap: 10px;
			padding: 15px;
			background-color: skyblue;
			margin: 0 35%;
		}
		.item-six {
			grid-column-start: 3;
			grid-column-end: 5;
		}
		.item-seven {
			grid-row-start: 2;
			grid-row-end: 4;
		}

Grid-items can be styled in the usual ways, but also can be sized to take up more or less space within a regular grid, as seen in the above example .item-six has been styled to take up two columns by identifying the left edge to right edge of the two column span and .item-seven spans two rows in a similar fashion.
