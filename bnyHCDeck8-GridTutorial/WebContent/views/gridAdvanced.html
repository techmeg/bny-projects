Grid Content

If you are familiar with the commands for controlling content in flex, the grid commands will look familiar. The justify-content and align-content properties are back, but with slightly different values. Center, space-around and space-between are the same as flex, but content controls include start, end, and the real outlier, >space-evenly.

Note: for the justify-content property to work, the grid's total width cannot exceed the size of the container.

Example: justify-content: center

This second grid is defined by justify-content: space-evenly; and the code looks like this: 

	.grid-container-space {
		display: inline-grid;
		width: 400px;
		grid-template-columns: auto auto;
		grid-gap: 10px;
		padding: 15px;
		text-align: center;
		background-color: skyblue;
		margin-bottom: 2%;
		justify-content: space-evenly;
	}

The text within the above grid has been styled using text-align: center; to center it within the grid box.
